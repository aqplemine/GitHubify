  • Webhook URL: {{WEBHOOK_URL}}

What works now

  • POST /webhook/github verifies X-Hub-Signature-256.
  • GET /auth/github/start begins OAuth.
  • GET /auth/github/callback handles code exchange when secrets are set.
  • GET /api/github/install-url returns install link.

Secrets stay server-side

Put your private key and client secret in environment variables only. Never expose them in frontend JavaScript.

Optional admin endpoint /api/github/app/installations requires x-admin-token.

GitHub-ify

Starter website for your GitHub App setup and webhook wiring.

Use these in your GitHub App settings

  • Homepage URL: {{APP_BASE_URL}}
  • Webhook URL: {{WEBHOOK_URL}}
  • Callback URL: {{CALLBACK_URL}}

Available endpoints

  • GET /health - health check
  • GET /app/config - returns app URLs in JSON
  • GET /auth/github/callback - OAuth callback placeholder
  • POST /webhook/github - GitHub webhook receiver with signature check
