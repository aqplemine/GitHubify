GitHub-ify

Starter website for your GitHub App setup and webhook wiring.

Use these in your GitHub App settings

  • Homepage URL: {{APP_BASE_URL}}
  • Webhook URL: {{WEBHOOK_URL}}
  • Callback URL: {{CALLBACK_URL}}

Available endpoints

  • GET /health - health check
  • GET /app/config - returns app URLs in JSON
  • GET /auth/github/callback - OAuth callback placeholder
  • POST /webhook/github - GitHub webhook receiver with signature check

Next steps

  • Deploy this service (Render, Railway, Fly.io, etc.).
  • Put your deployed HTTPS URL into GitHub App settings.
  • Set GITHUB_WEBHOOK_SECRET in both your app and GitHub App config.
  • Implement persistence for recommendations/favorites.
